# Lune Orchid - Official Website

A modern, responsive website for singer Lune Orchid featuring her music, upcoming events, and behind-the-scenes team information.

## Features

- **Hero Section**: Eye-catching landing page with gradient animations
- **About Section**: Personal story and musical journey
- **Music Section**: Showcase of songs and albums with play buttons
- **Events Section**: Upcoming concerts and performances with ticket links
- **Behind the Scenes**: Meet the creative team that brings the music to life
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Smooth Animations**: Modern UI with fade-in effects and hover interactions
- **Newsletter Signup**: Stay connected with fans

## Getting Started

Simply open `index.html` in your web browser to view the website. No build process or dependencies required!

## Customization

### Adding Real Images
Replace the placeholder divs with actual images:
- Update the `.image-placeholder` divs in the HTML with `<img>` tags
- Or add background images via CSS

### Adding Real Music
Connect the play buttons to actual audio players:
- Integrate with Spotify, Apple Music, or SoundCloud embeds
- Or implement a custom audio player

### Updating Content
All content is easily editable in `index.html`:
- Update the artist name, bio, and story
- Add real music tracks and albums
- Update event dates and locations
- Customize team member information

### Styling
Modify `styles.css` to change:
- Color scheme (update CSS variables in `:root`)
- Fonts (change Google Fonts imports)
- Layout and spacing
- Animations and effects

## Browser Support
